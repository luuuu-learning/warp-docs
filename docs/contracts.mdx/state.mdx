```rust
     create_job()
          │
          │◄─────────────┐
          │              │

      ┌───────┐          │
┌───► │Pending├─┬──► update_job()
│     └───┬───┘ │
│         │     │
│         │     └──► delete_job() ─┐
│         │                        │
│         │                        │
│         │                        ▼
│         ├─────► evict_job()  ┌─────────┐
│         │                    │Cancelled│
│         ▼           │        └─────────┘
└── execute_job()     ▼
                   ┌──────┐
          │        │Failed│
          │        └──────┘
          ▼
      ┌────────┐
      │Executed│
      └────────┘

```
