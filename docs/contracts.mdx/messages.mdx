# Execute messages

<CH.Scrollycoding>

## CreateJobMsg

Creates the job using a name, [condition](../condition.mdx), execute Wasm message, and a reward. Jobs are stored in an index. Successfully created jobs are given the _`Pending`_ status. 


```rust Rust
#[cw_serde]
pub struct CreateJobMsg {
    pub name: String,
    pub condition: Condition,
    pub msg: String,
    pub reward: Uint128,
}
```
